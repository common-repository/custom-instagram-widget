Custom Instagram Widget is very simple widget to showcase your latest instagram images. 

== Description ==

This widget allows you to showcase your recent pictures in a very simple way. No authentication required, just specify your username in the widget's form and you will get the images.

Markup is very simple. On clicking the images you will be directed to the instagram image detail page.

= Features =
* No Authentication required.
* Video type media can also be shown. But you will only get image having the video page link.
* Very *easy set up* process.
* Display latest photos from *your instagram account*.
* Manage Space between the images.



In case you face any problem, contact us via the [Forums](http://wordpress.org/support/plugin/custom-instagram-widget).

This is just a widget but if you want to show insatrgam feeds in the pages, need control on layout, image resolution, padding etc etc than you can try [Custom Instagram Feed Plugin](https://wordpress.org/plugins/easy-instagram-feed/)
== Installation ==

1. Download Custom Instagram Widget.
2. Upload the custom-instagram-widget folder to the /wp-content/plugins/ directory.
3. Activate the plugin through the 'Plugins' menu in WordPress and Enjoy.


== FAQ ==

= 1. How to display instagram feed on post / page? =
If you want to show feeds on the pages / posts than you can use our [Custom Instagram Feed Plugin](https://wordpress.org/plugins/easy-instagram-feed/)
